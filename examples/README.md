# examples — End-to-End Example Investigations

**Purpose:** Fully worked example cases (a realistic multi-evidence
investigation from upload through executive PDF report) used in the README
walkthrough, the User Guide (`docs/user-guide.md`), and demo recordings.
Distinct from `data/sample_evidence/` (raw parser fixtures): `examples/`
contains the *narrative*, expected findings, and final report for a complete
case, aimed at a human reader rather than a test runner.

**Future expansion:** A Jupyter notebook walkthrough of the LangGraph
investigation graph could live here once the graph is implemented.

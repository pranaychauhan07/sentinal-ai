# adr — Architecture Decision Records

One immutable file per significant decision, numbered sequentially
(`0001-...md`, `0002-...md`, ...). ADRs are never edited to reverse a decision —
a reversal is a *new* ADR that supersedes an old one (the old one is marked
"Superseded by ADR-00XX" but left intact). See `0000-template.md` for the
format and `docs/roadmap.md` for how ADRs map to milestones.
